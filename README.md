MiPádel es una aplicación móvil desarrollada en Android Studio con Kotlin y Jetpack Compose.
Su objetivo es facilitar la organización de partidos de pádel, permitiendo a los usuarios crear partidos, unirse a ellos, gestionar amigos, consultar estadísticas y mantener un historial de encuentros.
Este proyecto forma parte del Trabajo de Fin de Ciclo (DAM) e integra múltiples tecnologías modernas del ecosistema Android y Firebase.

Descripción del Proyecto
La aplicación permite:
- Registro e inicio de sesión mediante Firebase Authentication.

- Visualización de partidos activos y posibilidad de unirse a una posición libre.

- Creación de partidos a través de un formulario en un BottomSheet.

- Edición del perfil de usuario, incluyendo subida de imagen mediante ImgBB.

- Gestión de amistades: solicitudes, aceptación, rechazo y eliminación.

- Historial de partidos finalizados con cálculo de estadísticas.

- Persistencia de preferencias con DataStore (tema de la app y mantener sesión).

El sistema usa Firebase Firestore como base de datos principal y ImgBB como almacenamiento externo de imágenes.
